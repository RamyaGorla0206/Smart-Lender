# Smart Lender: Machine Learning Based Loan Approval Prediction System

Smart Lender is a B.Tech academic project developed under the APSCHE guidelines. It is a machine learning-powered web application designed to evaluate loan applicants' creditworthiness in real-time, assisting banking professionals in making data-driven loan decisions to minimize credit risks and non-performing assets (NPAs).

---

## Key Features
- **Exploratory Data Analysis (EDA):** Univariate, bivariate, and multivariate visualizations using Seaborn and Matplotlib.
- **Handling Imbalanced Classes:** Implements Synthetic Minority Over-sampling Technique (SMOTE) to balance applicant loan approval classes.
- **Feature Scaling:** Uses `StandardScaler` to normalize features for stable model optimization.
- **Model Comparisons:** Trains and evaluates four classifiers: Decision Tree, Random Forest, K-Nearest Neighbors (KNN), and XGBoost.
- **Best Model Integration:** Exports the trained XGBoost model (`rdf.pkl`) via Pickle.
- **Sleek Web Interface:** Built using Flask, Bootstrap 5, and custom glassmorphism styles, providing a responsive and modern user interface.
- **Cloud Deployment:** Complete blueprints and docker configurations for deployment on IBM Cloud Code Engine.

## Quick Start Guide

### 1. Prerequisites
Ensure you have Python 3.8 or higher installed on your system. 

### 2. Install Dependencies
Clone/copy the workspace directory and install requirements:
```bash
pip install -r requirements.txt
```

### 3. Generate the Dataset
Run the data generator to create the synthetic loan applications CSV file in the `data/` directory:
```bash
python data/generate_dataset.py
```

### 4. Train the Machine Learning Models
Execute the training script. This script processes the dataset, applies SMOTE, scales features, evaluates all four algorithms, prints accuracy comparisons, and saves the final XGBoost model to `rdf.pkl`:
```bash
python train.py
```

### 5. Launch the Web Application
Start the Flask web server locally:
```bash
python app.py
```
Open your browser and navigate to the local URL (typically [http://localhost:5000](http://localhost:5000)).
